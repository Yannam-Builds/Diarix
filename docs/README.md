# Documentation

This directory contains current Diarix product and engineering documentation.

- [`ALPHA_RELEASE.md`](ALPHA_RELEASE.md) — alpha scope, editions, release gate, and ship decision
- [`ARCHITECTURE.md`](ARCHITECTURE.md) — source boundaries and cross-layer invariants
- [`DEVELOPMENT.md`](DEVELOPMENT.md) — local setup and verification

The older Fumadocs application under this directory is inherited reference material and is excluded
from the active Bun workspace and alpha build. Do not add new Diarix documentation to that site.

Session handoffs and integration logs describe history rather than current truth. Use them only for
forensics; keep active decisions in the three documents above and in `.context/CONTEXT.md`.
